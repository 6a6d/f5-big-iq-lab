Module 6: Visibility for Legacy applications in the Applications tab (new 7.1)
==============================================================================

**[New 7.1.0]**

In BIG-IQ 7.1, the application dashboards support pre-deployed, “legacy” (or “brownfield”) 
configuration. These dashboards are read-only, except for enable, disable operations.

Fore more information about guideline regarding this new feature, please consult `K02142132`_.

.. _K02142132: https://support.f5.com/csp/article/K02142132
